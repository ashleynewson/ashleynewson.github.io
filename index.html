Various toys to play with

  • LineLogic 2 - Create logic circuits using an abstract and minimalistic set of rules.
  • LineLogic 2 WASM - LineLogic 2 with the heavy-duty parts re-written in Rust and compiled to WASM.
  • Morse Code Transcriber - Encode, decode, and practice listening to and keying in Morse code.
  • Tail Arena - A bullet-hell snake-like game.
  • Garbage Disposal Tycoon - An unfinished 3d isometric builder game.
  • Dissident Manager 2018 - Top-down stealth survival game. (collaboration)
  • Unnamed Ludum Dare 43 entry - Unfinished god sim with ray-casted terrain. (collaboration)
